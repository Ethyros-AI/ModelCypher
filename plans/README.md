# Plans Directory

This directory contains planning and audit documents.

## Status

| Document | Purpose | Status |
|----------|---------|--------|
| [comprehensive_audit_plan.md](comprehensive_audit_plan.md) | Audit methodology template | Template |
| [comprehensive_audit_report_structure.md](comprehensive_audit_report_structure.md) | Report structure template | Template |

These documents serve as templates for conducting systematic codebase audits. They are not active work items.

## Usage

When conducting a new audit:
1. Copy the relevant template
2. Fill in execution results
3. Archive completed audits with date suffix (e.g., `audit_2025-01-15.md`)
